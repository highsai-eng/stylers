## 動作確認
```shell script
git clone git@github.com:highsai-eng/stylers.git
cd stylers
```
or
```shell script
git clone https://github.com/highsai-eng/stylers.git
cd stylers
```
上記の後にこれする。
黒い画面に出てきたQRコードをスマホで写真撮る。
```shell script
yarn install
yarn start
```
